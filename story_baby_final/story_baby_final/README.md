## Deploy lên GitHub Pages
1. github.com → New repo → Public → Create
2. Add file → Upload files → kéo index.html + thư mục assets vào
3. Commit changes
4. Settings → Pages → Source: main / root → Save
5. Live tại: https://[username].github.io/[repo-name]/
